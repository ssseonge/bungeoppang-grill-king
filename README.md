# 붕어빵 굽기왕

로그인 없이 브라우저에 이름과 점수를 저장하는 모바일용 붕어빵 굽기 미니게임입니다.

## 플레이

- 철판 칸을 터치해서 `반죽 > 뒤집기 > 꺼내기`를 진행합니다.
- 아래 `펌핑! 화력증가!` 버튼을 눌러 화력을 올립니다.
- 재고가 있으면 손님이 자동으로 사갑니다.
- 손님을 놓치면 평판이 떨어지고, 평판이 0이 되면 게임오버입니다.

## 배포

정적 파일만 사용하므로 GitHub Pages에서 바로 실행할 수 있습니다.

```text
index.html
styles.css
game.js
BMJUA.woff2
```

점수와 이름은 서버가 아니라 현재 브라우저의 `localStorage`에 저장됩니다.
